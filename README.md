# UTN-TUPaD Programacion1

Proyecto práctico para la materia Programación I de la Tecnicatura Universitaria en Programación a Distancia.

## Descripción

Este repositorio contiene la práctica integradora de estructuras repetitivas y control de flujo en Python. El archivo principal `tp_integrador_repetitivas.py` implementa dos ejercicios:

- Ejercicio 1: manejo de la caja de un kiosco, con ingreso de datos de cliente, cantidad de productos, precio y descuento.
- Ejercicio 2: simulación de acceso seguro a un campus, con validación de usuario, clave y un menú protegido.

## Objetivo

Demostrar el uso de Git y GitHub Desktop mediante:

1. creación del repositorio local y remoto,
2. copia de los archivos de la práctica,
3. realización de commits claros y descriptivos,
4. envío de los cambios al repositorio remoto.

## Archivos

- `tp_integrador_repetitivas.py`: programa principal con las funciones y la lógica de los ejercicios.

## Ejecución

Para ejecutar el proyecto, abre una terminal en este directorio y ejecuta:

```bash
python tp_integrador_repetitivas.py
```

## Requisitos

- Python 3 instalado.

## Notas

Este proyecto forma parte de la entrega práctica de Git y GitHub de la unidad anterior de Programación I.
